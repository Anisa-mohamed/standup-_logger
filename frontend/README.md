# OneTwo Frontend

This React frontend connects to the existing backend API in `backend/`.

## Setup

1. Open a terminal in `frontend/`
2. Run `npm install`
3. Run `npm run dev`

The development server runs at `http://localhost:3001` and proxies API requests to `http://localhost:5000`.

## Backend

Start the backend from the repository root:

```bash
cd backend
python run.py
```

Then open the frontend at `http://localhost:3000`.
